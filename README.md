Steps to run locally 

1. **File Explorer:** Go to your desired folder.
2. **Terminal:** Open the folder in your terminal.
3. **Node Check:** Ensure you have Node installed (`node -v`). If not, install it first.
4. **Vite Setup:** Create a React app using `npm create vite@latest`.
5. **Configuration:** Select React and JavaScript based settings.
6. **Icons:** Run `npm i react-icons`.
7. **Cleanup:** Delete the default `src` folder.
8. **Navigation:** Open the terminal inside your project folder.
9. **Clone:** Run `git clone https://github.com/N1n4d0413/speedOtype1312.git`.
10. **Rename:** Rename the cloned folder `speedOtype1312` to `src`.
11. **Launch:** Run `npm run dev` in the terminal."# speedOtype1312"
